## 2. 技术架构决策

### 技术栈
| 组件 | 技术选择 | 理由 |
|------|----------|------|
| 前端框架 | Next.js 14 (App Router) | 最新技术、SSR支持、SEO友好 |
| UI框架 | Tailwind CSS + shadcn/ui | 快速开发、设计系统一致 |
| 后端框架 | Next.js API Routes | 全栈一体化、简化部署 |
| 数据库 | Supabase (PostgreSQL) | 实时数据库、身份验证、存储一体化 |
| 部署平台 | Vercel | Next.js最佳部署平台、全球CDN |
| AI服务 | Claude API (优先) | 计划文档推荐、长上下文、高质量输出 |
| 支付系统 | Stripe (海外) + 支付宝(国内) | 覆盖全球和国内用户 |
| 邮件服务 | Resend | 事务性邮件、开发者友好 |

### 开发环境
- **Node.js版本**：18.x LTS
- **包管理器**：npm / yarn / pnpm（根据项目配置）
- **代码规范**：ESLint + Prettier + TypeScript
- **版本控制**：Git + GitHub
